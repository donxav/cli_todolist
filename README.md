# To-Do List CLI

A simple command-line To-Do List application built using Python.

## Features

- Add tasks
- View tasks
- Remove tasks
- Input validation using try-except

## Concepts Practiced

- Functions
- Lists
- Loops
- Exception Handling
- User Input Validation

## How to Run

Clone the repository:

git clone https://github.com/donxav/cli_todolist.git

Move into the directory:

cd cli_todolist

Run:

python main.py
