# To-Do List CLI

A simple command-line To-Do List application built using Python.

## Features

- Add tasks
- View tasks
- Remove tasks
- Input validation using try-except

## Concepts Practiced

- Functions
- Lists
- Loops
- Exception Handling
- User Input Validation

## How to Run

Clone the repository:

``` bash
git clone  https://github.com/donxav/cli_todolist.git
```

Move into the directory:

```bash
cd cli_todolist
```

Run:

``` bash
python3 main.py
```
